GitHub 페이지 커스텀 도메인 설정하기

2024년 01월 30일
1 분 정도

깃허브 페이지로 블로그를 만들 때부터 개인 도메인으로 연결하고 싶다는 생각이 있었습니다. 이 글은 깃허브 페이지에서 커스텀 도메인 설정하며 정리한 내용입니다.

시작

기본적으로 깃허브 페이지를 만들면 아래의 URL 형태로 만들어 집니다.

https://깃허브아이디.github.io/

저의 경우는 https://plutslet0316.github.io/ 입니다.
위의 URL을 아래처럼 제 개인 도메인으로 변경할 생각입니다.

https://blog.plutslet.pe.kr/

도메인 DNS 설정

현재 사용 중인 DNS 관리 사이트에서 아래의 설정을 진행합니다.

  • 레코드 타입: CNAME
  • 서브 도메인(Name): 원하는 도메인 - blog
  • 대상 주소: 깃허브아이디.github.io

저는 클라우드플레어를 이용중이고 설정한 예시 이미지는 아래와 같습니다.

깃허브에서 커스텀 도메인 설정

  1. 원하는 저장소에서 Setting → Pages 을 차례로 클릭해 깃허브 페이지 설정창으로 이동합니다.
  2. Custom domain 항목을 찾아 입력란에 원하는 도메인을 입력 후 Save 버튼을 클릭합니다.
  3. 깃허브에서 DNS 확인을 진행하고, 이후 완료가 되면 해당 도메인으로 접근할 수 있게 됩니다.
